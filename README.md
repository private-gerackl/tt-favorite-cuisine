# tt-favorite-cuisine

## Архитектура
### Структура проекта
Корневая папка сервиса содержит конфигурационные файлы 
для используемых инструментов и папку с логикой app

app разделяется на 3 слоя:
* Транспортный - точки входа в логику сервиса
* Сервисный - бизнес-логика приложения
* Интеграционный - интерфейсы и реализация взаимодействия с интеграциями

Помимо этого там же находится папка meta, содержащая код, 
который нельзя однозначно отнести к одному из слоев,
и механизмы сборки внутри проектных зависимостей
