# tt-favorite-cuisine

## Архитектура
### Структура проекта
Корневая папка сервиса содержит конфигурационные файлы 
для используемых инструментов и папку с логикой app

app разделяется на 3 слоя:
* Транспортный (_**transports**_) - точки входа в логику сервиса
* Сервисный (_**services**_) - бизнес-логика приложения
* Интеграционный (_**integrations**_) - интерфейсы и реализация взаимодействия с интеграциями

Помимо этого там же находится папка meta, содержащая код, 
который нельзя однозначно отнести к одному из слоев,
и механизмы сборки внутри проектных зависимостей

### Управление проектом
Для управления проектом используется утилита just, в justfile описаны типовые команды для работы с проектом

Чтобы увидеть список доступных команд, нужно выполнить
```shell
just --list
```
